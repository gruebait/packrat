packrat
===
A flexible texture atlas creation tool.

## Planned Features

- **Generate game-ready texture atlases** from multiple standalone sources tiles and/or intermediate texture atlases in multiple formats.
- **Generate mipmaps with periodic downsampling** (no blurring between textures at higher mip levels).
- **Exotic atlas variations** including 4-tap textures, mirrored atlases, etc.
- **Texture adjustments** include padding and half-texel correction.
- **Command-line ready.**
